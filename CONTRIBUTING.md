# Contributing to gemhax

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo and clone locally
2. `cp .env.example .env` and add your Google API key
3. `pnpm i && pnpm dev`

## Development

- **Frontend** (`apps/web`): Next.js App Router + TypeScript
- **Backend** (`apps/api`): Express + TypeScript
- **Shared** (`packages/shared`): Zod schemas, types, prompt utilities

## Pull Requests

- Keep PRs focused and small
- Add tests for new functionality
- Run `pnpm test` before submitting
- Follow existing code style

## Issues

Open an issue for bugs, feature requests, or questions.
